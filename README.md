# MindMap

단축키 편의성에 초점을 맞춘 마인드맵 툴
MindNode에서 영감을 많이 받았지만, Windows에서는 이정도 편의성을 갖춘 툴이 없음. ㅠㅠ

## 단축키

- `Ctrl + S` : Save
- `Ctrl + O` : Open

## 마인드 맵 편집

노드를 마우스로 클릭하거나 키보드 방향키를 이용해 선택하고, 스페이스 바를 눌러 해당 노드의 내용을 수정할 수 있습니다.
(마인드 맵의 노드는 아이디어를 나타내는 정점을 말합니다.)

### 편집 단축키

노드를 선택한 상태에서
- space : 노드 편집 모드를 시작합니다. (편집 모드를 시작하면 내용을 수정할 수 있습니다.)
- 방향키 : 선택한 노드를 방향에 맞는 다른 노드로 변경합니다.
- `ctrl + ←`, `ctrl + →` : 선택한 노드를 부모/자식 노드로 옮깁니다. 루트 노드의 바로 다음 자식인 경우, 왼쪽/오른쪽 그룹으로 변경합니다.
- `ctrl + ↑`, `ctrl + ↓` : 선택한 노드를 형제와 순서를 바꿉니다.
- `ctrl + v` : 복사한 이미지를 붙여넣습니다. (텍스트 및 노드는 향후 지원 예정입니다.)
- `ctrl + c` (복사는 향후 지원 예정입니다.)

노드를 편집 중인 상태에서
- esc : 노드 편집 모드를 마칩니다. (편집 모드를 마치면 내용을 수정할 수 없습니다.)
